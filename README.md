# ShopNest

ShopNest is a full-stack e-commerce project with:

- a React frontend
- an Express and Node.js backend
- MongoDB storage
- OTP-based email verification

## Project structure

- frontend: React app
- backend: Node.js/Express API

## Run locally

### Backend

```bash
cd backend
npm install
node index.js
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Notes

The frontend uses a local API proxy for development. For production deployment, the frontend API URL should be updated to the deployed backend URL.
